## Danish regulation

- **Momsloven** (Danish VAT Act) — particularly §§69–71 (second-hand goods margin scheme / *brugtmomsordningen*).

- **DMR / Motorregister** — the Danish Motor Vehicle Register; authoritative for vehicle identity and registration status.
- **Afregningsbilag** rules — self-billing settlement notes for purchases from non-VAT-registered sellers.
